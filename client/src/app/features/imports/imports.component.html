Payload Fields — Single Record

FieldTypeRequiredDescription
apiKey string Yes Your store API key — also determines which store the record belongs to
productId string Yes Unique identifier for the product (maps to ProductId in Analytics)
price number (2 dec.) Yes Unit price of the product (maps to Price)
quantitySold integer Yes Number of units sold in this record (maps to QuantitySold)
stock integer Yes Current stock remaining (maps to Stock)

Payload Fields — Bulk

FieldTypeRequiredDescription
apiKey string Yes Your store API key
items array Yes List of records — each with productId, price, quantitySold, stock
